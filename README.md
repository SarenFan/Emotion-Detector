# Emotion Detection Application

## Project Overview
This project is an AI-based web application that detects emotions from text using the Watson NLP library. It was developed as a final project to demonstrate skills in API integration, web deployment, and software engineering practices.

## Features
* Extracts emotions (anger, disgust, fear, joy, sadness) from user-provided text.
* Identifies the dominant emotion in the given text.
* Web interface built and deployed using Flask.
* Includes unit testing, error handling for blank inputs, and static code analysis to ensure code quality.

## Technologies Used
* Python
* Flask
* Watson NLP Library
